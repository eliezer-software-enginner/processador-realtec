# Processador Realtec

Eu realizei o desenvolvimento de um aplicativo desktop e é bem simples de utilizar.
Utilizei JavaFX juntamente com meu framework próprio Megalodonte.
## Primeiramente realize o download da aplicação
[Fazer download](https://www.mediafire.com/file/pq17z98cdikd4ff/realtec-processador.zip/file)

Após download, extraia o arquivo e execute o arquivo `realtec-processador.exe` clicando duas vezes sobre ele.
Então você já poderá seguir com o uso do aplicativo normalmente.
